Java String Methods

length()

  1. public int length()
  2. Returns the length of a specified string
  3. Return type is int

Example:

        public class Solution 
        {
            public static void main(String[] args) 
            {
                String str = "Karan";
                System.out.println(str.length()); // 5
            }
        }

charAt()

  1. public char charAt(int index)
  2. Returns the character at the specified index (position)
  3. Return type is char

Example:

        public class Solution 
        {
            public static void main(String[] args) 
            {
                String str = "Karan";
                char ch = str.charAt(0);

                System.out.println(ch);               // K
                System.out.println(str.charAt(1));    // a
                System.out.println(str.charAt(2));    // r
                System.out.println(str.charAt(3));    // a
                System.out.println(str.charAt(4));    // n
                // System.out.println(str.charAt(5)); // IndexOutOfBoundsException
            }
        }

concat()

  2. Appends a string to the end of another string

Example:

        public class Solution

            public static void main(String[] args) 
            {

compareTo()

  1. public int compareTo(String string2)
    public int compareTo(Object object)
  2. Compares two strings lexicographically
  3. Return type is int
    • = 0 if the string is equal to the other string
    • < 0 if the string is lexicographically less than the other string
    • > 0 if the string is lexicographically greater than the other string (more characters)

equals()

  1. public boolean equals(Object anotherObject)
  2. Compares two strings. Returns true if the strings are equal, and false if not
  3. Return type is boolean
    • true - if the strings are equal
    • false - if the strings are not equal

equalsIgnoreCase()

  1. public boolean equalsIgnoreCase(String anotherString)
  2. Compares two strings, ignoring case considerations
  3. Return type is boolean
    • true - if the strings are equal, ignoring case differences
    • false - if the strings are not equal

Example:

        public class Solution 
        {
            public static void main(String[] args) 
            {
                String str1 = "Hello", 
                str2 = "Hello", 
                str3 = "Hi", 
                str4 = "Karan", 
                str5 = "KaraN";      

                int value = str4.compareTo(str5);    

                System.out.println(str1.compareTo(str2));          // 0
                System.out.println(str1.compareTo(str3));          // -4
                System.out.println(value);                         // 32
                System.out.println(str5.compareTo(str4));          // -32
                    
                System.out.println(str1 == str2);                  // true
                System.out.println(str1 == str3);                  // false
                System.out.println(str4 == str5);                  // false
                System.out.println(str5 == str4);                  // false
                            
                System.out.println(str1.equals(str2));             // true
                System.out.println(str1.equals(str3));             // false
                System.out.println(str4.equals(str5));             // false
                System.out.println(str5.equals(str4));             // false

                System.out.println(str1.equalsIgnoreCase(str2));   // true
                System.out.println(str1.equalsIgnoreCase(str3));   // false
                System.out.println(str4.equalsIgnoreCase(str5));   // true
                System.out.println(str5.equalsIgnoreCase(str4));   // true
            }
        }

indexOf()

  1. int indexOf(char ch)
    int indexOf(String str)
    int indexOf(char ch, int start)
    int indexOf(String str, int start)
  2. Returns the position of the first found occurrence of specified characters in a string
  3. An int value, representing the index of the first occurrence of the character in the string, or -1 if it never occurs
  4. Return type is String

Example:

        public class Solution 
        {
            public static void main(String[] args) 
            {
                String str = "Karan Suthar";

                System.out.println(str.indexOf('K'));            // 0
                System.out.println(str.indexOf('a'));            // 1
                System.out.println(str.indexOf('r'));            // 2
                System.out.println(str.indexOf('a'));            // 1
                System.out.println(str.indexOf('N'));            // -1

                System.out.println(str.indexOf("Karan"));        // 0
                System.out.println(str.indexOf("Suthar"));       // 6
                System.out.println(str.indexOf("Karan Suthar")); // 0 
                System.out.println(str.indexOf("aran"));         // 1
                System.out.println(str.indexOf("ran"));          // 2
                System.out.println(str.indexOf("an"));           // 3
                System.out.println(str.indexOf("n S"));          // 4

                System.out.println(str.indexOf('a', 0));         // 1            
                System.out.println(str.indexOf('a', 3));         // 3
                System.out.println(str.indexOf('a', 4));         // 10

                System.out.println(str.indexOf("Suthar", 3));    // 6       
                System.out.println(str.indexOf("Suthar", 6));    // 6
                System.out.println(str.indexOf("Suthar", 7));    // -1
            }
        }

replace()

  1. public String replace(CharSequence target, CharSequence replacement)

replaceFirst()

  1. public String replaceFirst(String regex, String replacement)
  2. Replaces the first occurrence of a substring that matches the given regular expression with the given replacement
  3. Return type is String

replaceAll()

  1. public String replaceAll(String regex, String replacement)
  2. Replaces each substring of this string that matches the given regular expression with the given replacement
  3. Return type is String

Example:

        public class Solution 
        {
            public static void main(String[] args) 
            {
                String str = "Karan Suthar";

                String newStr1 = str.replace('a', 'i');
                System.out.println(newStr1);    // Kirin Suthir
                System.out.println(str);        // Karan Suthar

                String newStr6 = str.replace("ar", "al");
                System.out.println(newStr6);    // Kalan Suthal
                System.out.println(str);        // Karan Suthar

                String newStr2 = str.replaceFirst("a", "i");
                System.out.println(newStr2);    // Kiran Suthar
                System.out.println(str);        // Karan Suthar

                String newStr3 = str.replaceAll("a", "i");
                System.out.println(newStr3);    // Kirin Suthir
                System.out.println(str);        // Karan Suthar

                String newStr4 = str.replaceFirst("ar", "al");
                System.out.println(newStr4);    // Kalan Suthar
                System.out.println(str);        // Karan Suthar

                String newStr5 = str.replaceAll("ar", "al");
                System.out.println(newStr5);    // Kalan Suthal
                System.out.println(str);        // Karan Suthar
            }
        }
