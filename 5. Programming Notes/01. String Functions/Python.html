Python String Methods

String Length

  1. len(object)
  2. The len() function returns the number of items in an object
  3. When the object is a string, the len() function returns the number of characters in the string

Example:

        str = "Karan"
        print(len(str))   # 5   

        str = " Karan Suthar "
        print(len(str))   # 14

Accessing Character

Example:

        str = "Karan Suthar"

        print(str[2])  # r
        print(str[3])  # a
        print(str[4])  # n
        print(str[6])  # S

        print(str[-2]) # a
        print(str[-3]) # h
        print(str[-4]) # t
        print(str[-6]) # S

String Concatenation

Example:

        str1 = "Karan"
        str2 = "Suthar"

        newStr = str1 + " " + str2

        print(newStr) # Karan Suthar

String Comparison

Example:

        str1 = "Karan";
        str2 = "KaraN";

        print(str1 > str2)   # True
        print(str1 < str2)   # False
        print(str1 == str2)  # False

index()

  1. string.index(value, start, end)
  2. The index() method finds the first occurrence of the specified value
  3. The index() method raises an exception if the value is not found

find()

  1. string.find(value, start, end)
  2. The find() method finds the first occurrence of the specified value
  3. The find() method returns -1 if the value is not found

Example:

        str = "Karan Suthar"

        print(str.index('r'))              # 2
        print(str.index(" S"))             # 5
        # print(str.index(' K'))           # ValueError: substring not found 

        print(str.find('r'))               # 2
        print(str.find(" S"))              # 5
        print(str.find(' K'))              # -1

        # print(str.index("ran", 0, 4))    # ValueError: substring not found
        print(str.index("ran", 0, 5))      # 2

        print(str.find("ran", 1, 4))       # 1
        print(str.find("ran", 1, 5))       # 2

replace()

  1. string.replace(oldValue, newValue, count)
  2. Returns a string where a specified value is replaced with a specified value

Example:

        str = "Karan Suthar"

        print(str)                       # Karan Suthar
        print(str.replace("ran", 'run')) # Karun Suthar
        print(str.replace("a", "i", 1))  # Kiran Suthar
        print(str.replace('a', 'i', 2))  # Kirin Suthar
        print(str.replace("a", "i"))     # Kirin Suthir

Substring

Example:

        str = "Karan Suthar"

        print(str[2:5])     # ran    
        print(str[-10:-7])  # ran  
        print(str[2:-2:3])  # r t  
        print(str[::])      # Karan Suthar          
        print(str[::2])     # KrnSta           
        print(str[2::])     # ran Suthar           
        print(str[:-2:])    # Karan Suth 
        print(str[2:500])   # ran Suthar
        print(str[-343:5])  # Karan

lower()

  1. string.lower()
  2. The lower() method returns a string where all characters are lower case
  3. Symbols and Numbers are ignored

upper()

  1. string.upper()
  2. The upper() method returns a string where all characters are in upper case
  3. Symbols and Numbers are ignored

capitalize()

  1. string.capitalize()
  2. The capitalize() method returns a string where the first character is upper case, and the rest is lower case

title()

  1. string.title()
  2. The title() method returns a string where the first character in every word is upper case. Like a header, or a title
  3. If the word contains a number or a symbol, the first letter after that will be converted to upper case

swapcase()

  1. string.swapcase()
  2. The swapcase() method returns a string where all the upper case letters are lower case and vice versa

Example:

        str = "kaRan sUthAR 453jI"

        newStr1 = str.lower()
        newStr2 = str.upper()
        newStr3 = str.capitalize()
        newStr4 = str.title()
        newStr5 = str.swapcase()

        print(newStr1) # karan suthar 453ji 
        print(newStr2) # KARAN SUTHAR 453JI 
        print(newStr3) # Karan suthar 453ji 
        print(newStr4) # Karan Suthar 453Ji 
        print(newStr5) # KArAN SuTHar 453Ji

lstrip()

  1. string.lstrip(characters)
  2. Returns a left trim version of the string

rstrip()

  1. string.rstrip(characters)
  2. Returns a right trim version of the string

strip()

  1. string.strip(characters)
  2. Returns a trimmed version of the string

Example:

        txt1 = "    Karan    Suthar    "

        str1 = txt1.lstrip();
        str2 = txt1.rstrip();
        str3 = txt1.strip();
        
        print(str1)  # Karan    Suthar    
        print(str2)  #     Karan    Suthar
        print(str3)  # Karan    Suthar
        
        txt2 = ",,,,,nnttgg...    ..banana...   ,,,,,   .ngtn"
        
        str4 = txt2.lstrip(",. ntg")
        str5 = txt2.rstrip(" .nt,g")
        str6 = txt2.strip(" .g,tn")
        
        print(str4)  # banana...   ,,,,,   .ngtn
        print(str5)  # ,,,,,nnttgg...    ..banana
        print(str6)  # banana

split()

  1. string.split(separator, maxsplit)
  2. The split() method splits a string into a list
  3. You can specify the separator, default separator is any whitespace
  4. When maxsplit is specified, the list will contain the specified number of elements plus one

Example:

        str = "Hello Jii, Kese Hain Aap, Sab Badhiya"

        lst1 = str.split()
        lst2 = str.split(",")
        lst3 = str.split(",", 1)

        print(lst1) # ['Hello', 'Jii,', 'Kese', 'Hain', 'Aap,', 'Sab', 'Badhiya']
        print(lst2) # ['Hello Jii', ' Kese Hain Aap', ' Sab Badhiya']
        print(lst3) # ['Hello Jii', ' Kese Hain Aap, Sab Badhiya']

zfill()

  1. string.zfill(len)
  2. The zfill() method adds zeros (0) at the beginning of the string, until it reaches the specified length
  3. If the value of the len parameter is less than the length of the string, no filling is done

Example:

        a = "25"
        b = "Karan"
        c = "19.456"
        d = "Welcome to the Jungle"
            
        print(a.zfill(10)) # 0000000025
        print(b.zfill(10)) # 00000Karan
        print(c.zfill(10)) # 000019.456
        print(d.zfill(10)) # Welcome to the Jungle
